화장실 청소하는 방법

화장실 청소하는 방법

목차

1. 준비물 2. 화장실 청소 순서 3. 화장실 청소 도구 4. 화장실 청소 주기 5. 화장실 청소 팁

1. 준비물

* 청소용품 * 락스 * 베이킹소다 * 구연산 * 스테인리스 클리너 * 유리세정제 * 청소솔 * 스펀지 * 마른 수건 * 장갑 * 마스크 * 고무장갑

2. 화장실 청소 순서

1. 배수구 청소 * 배수구 커버를 꺼내고 머리카락과 이물질을 제거한다. * 베이킹소다와 뜨거운 물을 1:1 비율로 섞어 배수구에 부어준다. * 20분~30분 후 뜨거운 물을 다시 부어준다. 2. 변기 청소 * 변기 변좌에 락스를 뿌린다. * 10분 후 변기 브러시로 변기를 닦는다. * 깨끗한 물로 변기를 헹군다. 3. 세면대 청소 * 세면대에 베이킹소다를 뿌린다. * 10분 후 스펀지로 세면대를 닦는다. * 깨끗한 물로 세면대를 헹군다. 4. 욕조 청소 * 욕조에 구연산수를 뿌린다. * 10분 후 스펀지로 욕조를 닦는다. * 깨끗한 물로 욕조를 헹군다. 5. 거울 청소 * 거울에 유리세정제를 뿌린다. * 마른 수건으로 거울을 닦는다. 6. 타일 청소 * 타일에 락스를 뿌린다. * 10분 후 스펀지로 타일을 닦는다. * 깨끗한 물로 타일을 헹군다.

3. 화장실 청소 도구

* 락스 * 베이킹소다 * 구연산 * 스테인리스 클리너 * 유리세정제 * 청소솔 * 스펀지 * 마른 수건 * 장갑 * 마스크 * 고무장갑

4. 화장실 청소 주기

* 변기 : 일주일에 2회 * 세면대 : 일주일에 1회 * 욕조 : 일주일에 1회 * 거울 : 매일 * 타일 : 한 달에 1회

5. 화장실 청소 팁

* 청소할 때는 반드시 장갑과 마스크를 착용한다. * 청소용품은 용도에 맞게 사용한다.
